# Portfolio notes

This repository is designed to look like a realistic data engineering portfolio project.

## Story
- Build a small telemetry stack locally.
- Capture metrics into InfluxDB.
- Visualize them with Grafana.
- Add an OBD2 integration path for vehicle telemetry experiments.

## Why it is useful
- It demonstrates containerization.
- It shows time-series data handling.
- It highlights dashboarding and observability.
- It provides a path to expand into real vehicle data ingestion.
